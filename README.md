# Snippets App Vault

This is a backup and storage vault for all my notes that I wrote in the Snippets app.

## Purpose

This repository serves as a secure backup location for notes and snippets, providing:
- Version control for all notes
- Cloud storage backup
- Easy restoration when needed
- Organized structure for categorizing notes

## Structure

```
notes/
├── personal/       # Personal notes and thoughts
├── work/          # Work-related notes
├── code/          # Code snippets and examples
└── misc/          # Miscellaneous notes
```

## Usage

### Backing Up Notes

1. Add your notes to the appropriate category folder
2. Commit your changes with descriptive messages
3. Push to the repository

### Restoring Notes

1. Clone or pull the latest version of this repository
2. Copy the notes you need from the appropriate folder
3. Import them back into your Snippets app
